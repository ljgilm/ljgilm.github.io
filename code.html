Coding Projects

Bloodhound

Created a mobile app in Mobile Application Development course. The app focuses on creating a space where users can post and easily keep track of lost and found dogs. Proposed the idea after seeing the aftermath of Hurricane Ida. I also was inspired by my dog Heidi who is a resuce from one of the many floods that occur in Louisiana.

I led the project and worked and coded with teammates to create an app. We were very proud of what we accomplished especially with the amount of time we lost because of Hurricane Ida. We did cut some features such as connecting the app to a database and having multiple users. However, we were able to prioritize the main features that we thought were important.

React Native, JSX

Shopping for Products Organizer

When deciding to purchase something online, there are so many options to choose from and keeping track of all them can be overwhelming. For me, I always end up having lots of tabs open to the point its unmanageable. Therefore, this program allows users to insert the different versions of a product they are interested in and then filter and sort them based on their preferences. An example could be wireless earbuds.

However, this program is not limited to physical items. Other examples could also include different digital art programs, streaming services, job opportunites, etc. The goal of this program is to make it easier for consumers to organize all their options so that they can make the best decision.

Java
